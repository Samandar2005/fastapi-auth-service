# Contributor Covenant Code of Conduct

Our project follows the Contributor Covenant code of conduct. By participating, you are expected to uphold this code.

Summary:
- Be respectful and considerate.
- Be open to constructive feedback.
- Harassment or abusive behavior will not be tolerated.

For the full text and reporting guidelines, see: https://www.contributor-covenant.org/version/2/0/code_of_conduct/
